# launchatdawn

High-performance brand site for launchatdawn, built with Next.js, Tailwind CSS, Framer Motion, and Sanity.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Sanity CMS

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Available Scripts

- `npm run dev` - start development server
- `npm run build` - build for production
- `npm run start` - run production build
- `npm run lint` - run ESLint

## Content Management

Sanity Studio is available in-app at:

- `/studio`

Make sure your Sanity environment values are configured before publishing content.

## Deployment

Deploy on Vercel (recommended), then connect the repository and configure environment variables for Sanity.
